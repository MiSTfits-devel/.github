# Who are we at MiSTfits?
Our devs hack on FPGA cores for:
- GBA
- N64
- PSX
- WonderSwan
- and more!

We add features other core devs don't have yet, and they'll copy from us:
- GBA accuracy improvements
  - GBA2P link support for most titles
  - The Shrek Video mapper (yes it's real)
  - Various memory and timing issues
- New hardware implementations
  - N64 DD support
  - PSX upgrades
    - serial and multitap support/enhancements
    - CD-ROM seek accuracy with funky new model
  - General WonderSwan improvements

## How can I get involved?
Fork the repos and get started! PRs welcome, and we're open to large reorganizations if well done
